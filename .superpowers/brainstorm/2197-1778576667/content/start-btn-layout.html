Реалтаймовая сортировка во время теста

Какое поведение предпочтительно?

C

Сортировка в реальном времени

Каждая завершённая стратегия сразу всплывает наверх по %. Список «живёт» — видишь лучших прямо сейчас. Немного визуального хаоса.

D

Сортировка только в конце

Во время теста: завершённые стратегии поднимаются выше незавершённых. Финальная сортировка по % — только когда всё закончилось. Стабильнее для чтения.

E

Завершённые сначала + % в конце

Во время теста: завершённые > незавершённые (как в оригинальном ByeDPI). После всех — дополнительная сортировка по %. Рекомендую этот вариант.

★ Как в оригинальном ByeDPI (вариант E):
✓ 67% · -Ku -An -d4 · avg 312ms [завершена первой]
✓ 0% · -s1 -q1 -a1 -Y · avg 0ms [завершена, провал]
⟳ 33% · -s5 -o2 -At ... · тестируется сейчас
○ -f-1 -r1+s ... [ожидание]

Кликни варианты выше (можно несколько) или просто напиши в терминале
