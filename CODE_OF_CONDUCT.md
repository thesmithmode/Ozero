# Code of Conduct

## Обещание

Цель проекта — дать пользователям в цензурированных юрисдикциях инструмент для свободного доступа к информации. Чтобы этого достичь — нам нужна сообщество, где люди разных опытов, навыков и происхождений могут вносить вклад без страха.

## Стандарты

**Ожидаемое**:
- Уважительный и профессиональный тон
- Technical critique — по сути, не по личности
- Открытость к разным точкам зрения
- Принятие конструктивной обратной связи
- Признание ошибок и извлечение уроков

**Неприемлемое**:
- Оскорбления, унижения, личные атаки
- Harassment публично или приватно
- Публикация чужой приватной информации без согласия
- Троллинг, namecalling
- Любой контент, нарушающий законы юрисдикций mainstream-сообщества (OFAC-sanctioned material, CSAM, terrorism incitement)

## Scope

Применяется в:
- GitHub issues, PRs, discussions
- Telegram группе Ozero (pending)
- Любых каналах где представители проекта действуют от имени Ozero

## Enforcement

Maintainer имеет право:
- Удалять комментарии / commits / commits / PRs нарушающие CoC
- Banить участников временно или постоянно

Сообщения о нарушениях — в issue с тэгом `code-of-conduct` или напрямую maintainer (GitHub DM / email).

## Особый пункт — technical rigor

Technical strictness — это НЕ нарушение CoC. Если ваш PR отклонён с жёсткой технической критикой — это нормально. Высокая планка качества критична для security-sensitive проекта.
