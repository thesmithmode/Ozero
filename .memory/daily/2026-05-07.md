**Lessons Learned:**

- Два Go runtime (am-go + ur-go) в одном процессе крашатся при rapid stop/start → eager-load обоих на старте, живут весь lifecycle
- Mirror auto-config DNS (`176.99.11.77, 80.78.247.254`) — не Cloudflare, подозрительные
- `awgTurnOn OK` + state=Connected не значит реальный handshake — нужен polling `last_handshake_time_sec`
