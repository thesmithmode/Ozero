- Сформирован полный план по ByeDPI, IP/FPTN, sing-box, MasterDNS, split tunnel и UI.

**Decisions Made:**
- Изолировать ByeDPI strategy evaluator от production engine, добавить hard-timeout и `stop()` в `NonCancellable finally`.

- PR делать только в `dev`; `main` не трогать без отдельной команды.

**Lessons Learned:**
- Общий singleton native ByeDPI между scan и VPN создаёт race window и crash-риск.
- UI “создание популяции” скрывает долгую первую оценку generation.
- Token-level GA создаёт невалидные ByeDPI argv и лишний native churn.
- Split tunnel не видит новые приложения из-за вечного cache `DefaultAppListProvider`.
- FPTN теряет флаг из-за неполного `countryCode` contract.

**Action Items:**
- Реализовать изменения по плану, добавить sentinel/unit tests, запускать полную валидацию только в GitHub Actions до terminal status.
- Проверить N>0 tests, coverage gate и создать PR в `dev` после зелёного CI.
