## Concepts

### V1 signing обязателен для максимальной совместимости

Android V2+V3-only signing технически корректно для API 24+, но часть кастомных ROM (MIUI, ColorOS, некоторые MediaTek-форки) требуют V1 JAR-подпись. Без неё установка даёт "пакет недействителен". Стоимость включения V1: +незначительный overhead в META-INF. Оставлять V1=true навсегда.

### DEX assertions must cover all manifest-declared custom classes

AGP auto-generates keep rules для manifest-компонентов, но R8 может всё равно их убрать при определённых конфигурациях. CI release assertions должны явно проверять каждый `android:name` в manifest — иначе "blind spot".
