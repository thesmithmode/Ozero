- При squash merge на свежий dev с разошедшимися `.memory/` правками — stash pop конфликтует. Решение: `git checkout --theirs` на conflicted .memory + drop stash, daily вести из текущей сессии. Старые правки в стэше теряются — приемлемо если контент менее ценный чем чистый merge.
